We can describe the set operations from the videos today using set builder notation, along with notation from unit 1. For example, for arbitrary sets \(A\) and \(B\) within a universal set \(\mathcal{U}\text{,}\) we can write

\begin{equation*} A\cup B = \bigl\{ x\in\mathcal{U} \; \colon\; x\in A \; \vee \; x\in B \bigr\}. \end{equation*}
