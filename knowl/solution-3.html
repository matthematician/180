Solution 1.2.1.3.1.

Assumptions: Suppose that

\begin{align} x&\in \mathbb{Z}\tag{1.2.8}\\ \text{but that } 4 &\mid (x^2+2).\tag{1.2.9} \end{align}

By definition of divisibility by 4, statement (1.2.9) implies the existence of an integer \(n\in\mathbb{Z}\) so that \(x^2 + 2 = 4n\text{.}\)

Because we assumed \(x\) was an integer ((1.2.8)), we can investigate this assertion by cases, depending on the divisibility of \(x\) by 4.

Case 1: If \(x\) is a multiple of 4. Then there exists an integer \(a\in\mathbb{Z}\) so that \(x=4a\text{.}\) But then

\begin{align*} x^2 + 2 &= (4a)^2 + 2\\ &= 16a^2 + 2\\ &=4\bigl(4a^2\bigr) + 2 \end{align*}

and since \(j=4a^2\) is an integer, we have \(x^2 + 2 = 4j + 2\text{.}\) Thus \((x^2+2)\) cannot be a multiple of \(4\text{,}\) since its remainder when divided by 4 is 2.

Case 2: If \(x\) is one more than a multiple of 4. Then there exists an integer \(b\in\mathbb{Z}\) so that \(x=4b+1\text{.}\) But then

\begin{align*} x^2 + 2 &= (4b+1)^2 + 2\\ &= 16b^2 + 8b +1 2\\ &=4\bigl(4b^2+2b\bigr) + 3 \end{align*}

and since \(k=4b^2+2b\) is an integer, we have \(x^2 + 2 = 4k + 3\text{.}\) Thus \((x^2+2)\) cannot be a multiple of \(4\text{,}\) since its remainder when divided by 4 is 3.

Case 3: If \(x\) is two more than a multiple of 4. Then there exists an integer \(c\in\mathbb{Z}\) so that \(x=4c+2\text{.}\) But then

\begin{align*} x^2 + 2 &= (4c+2)^2 + 2\\ &= 16c^2 +16c + 4+ 2\\ &=4\bigl(4c^2+4c+1\bigr) + 2 \end{align*}

and since \(\ell=4c^2+4c+1\) is an integer, we have \(x^2 + 2 = 4\ell + 2\text{.}\) Thus \((x^2+2)\) cannot be a multiple of \(4\text{,}\) since its remainder when divided by 4 is 2.

Case 4: If \(x\) is three more than a multiple of 4. Then there exists an integer \(d\in\mathbb{Z}\) so that \(x=4d+3\text{.}\) But then

\begin{align*} x^2 + 2 &= (4d+3)^2 + 2\\ &= 16d^2 + 24d + 9 + 2\\ &=4\bigl(4d^2+6d+2\bigr) + 3 \end{align*}

and since \(m=4d^2+6d+2\) is an integer, we have \(x^2 + 2 = 4m + 3\text{.}\) Thus \((x^2+2)\) cannot be a multiple of \(4\text{,}\) since its remainder when divided by 4 is 3.

Conclusion: Since our assumptions necessitate a contradiction, we conclude by way of contradiction that if \(x\in\mathbb{Z}\text{,}\) then \(4\nmid (x^2+2)\text{.}\)
