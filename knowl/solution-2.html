Solution 1.2.1.2.1.

Proof. Let \(a,b,c\in\mathbb{Z}\) be arbitrarily chosen.

Assumptions. Suppose that

\begin{align} a^2+b^2&=c^2\tag{1.2.3}\\ \text{but that both }a,b& \text{ are odd}.\tag{1.2.4} \end{align}

(See note  2 .)

By definition of odd, there exist integers \(k,\ell\in\mathbb{Z}\) such that \(a=2k+1\) and \(b=2\ell+1\text{.}\)

By (1.2.3) we have

\begin{align*} c^2 &= a^2+b^2\\ &= (2k+1)^2 + (2\ell+1)^2\\ &=4k^2+4k+4\ell^2+4\ell+2\\ &=2\bigl[ 2(k^2+k+\ell^2+\ell) + 1 \bigr]. \end{align*}

So on one hand, \(c^2\) is a perfect square. On the other, there exists an integer \(n = k^2+k+\ell^2+\ell\) so that \(c^2 = 2\bigl[ 2n + 1 \bigr]\text{.}\) Setting \(m = 2n+1\) we see that \(c^2 = 2m\) where \(m\) is an odd number.

Thus we know three things about \(c^2\text{:}\)

\begin{align} c^2 & \text{ is a perfect square}\tag{1.2.5}\\ c^2 &= 2m \text{ is an even number...}\tag{1.2.6}\\ &\text{ ...and }m\text{ is odd. }\tag{1.2.7} \end{align}

We proceed by cases.

Case 1: If \(c\) is odd. As we have shown previously, if \(c\) is odd then \(c^2\) is also odd. However this is a contradiction of (1.2.5), so this case cannot be sustained.

Case 2: If \(c\) is even. Then there exists an integer \(r\) such that \(c = 2r\text{.}\) But then

\begin{align*} c^2 &= (2r)^2\\ &= 4\, r^2 \end{align*}

and (1.2.6) implies that

\begin{equation*} 2m = 4r^2. \end{equation*}

Hence \(m = 2\, r^2\text{.}\) However, since \(r^2\) is an integer this implies that \(m\) is even, contradicting (1.2.7), so this case cannot be sustained.

Conclusion: Since our assumptions necessitate a contradiction, we conclude by way of contradiction that if \(a^2+b^2=c^2\) then \(a\) is even or \(b\) is even. \(\square\)
