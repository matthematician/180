Warmup Exercise 2.1.1.

Another way to describe a set is by defining some property all elements of the set have in common. We call this method set-builder notation.

\begin{equation*} \bigl\{ x \; \colon \; x \text{ satisfies some property} \bigr\} \end{equation*}

For example, \(A = \bigl\{ x \colon x \text{ is odd and }x\text{ is positive}\bigr\}\) can also be described

\begin{align*} A \amp= \{1,3,5,7,\ldots,\} \\ \amp= \{x\colon x=2y-1 \text{ for some }y\in\mathbb{N}\} \\ \amp= \text{the set of all odd positive integers}. \end{align*}

  1. Describe the members of the set

    \begin{equation*} \{ x\in\mathbb{Z} \colon x=3k \text{ for some }k\in\mathbb{Z} \}. \end{equation*}

  2. Describe the set of all even natural numbers that are less than 100 with set-builder notation.

  3. Describe the following set in set-builder notation:

    \begin{equation*} \{ -8,-7,-6,\ldots,10\}. \end{equation*}
