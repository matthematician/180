Section 2.1 Sets and Subsets

Warmup Exercise 2.1.1.

Another way to describe a set is by defining some property all elements of the set have in common. We call this method set-builder notation.

\begin{equation*} \bigl\{ x \; \colon \; x \text{ satisfies some property} \bigr\} \end{equation*}

For example, \(A = \bigl\{ x \colon x \text{ is odd and }x\text{ is positive}\bigr\}\) can also be described

\begin{align*} A \amp= \{1,3,5,7,\ldots,\} \\ \amp= \{x\colon x=2y-1 \text{ for some }y\in\mathbb{N}\} \\ \amp= \text{the set of all odd positive integers}. \end{align*}

  1. Describe the members of the set

    \begin{equation*} \{ x\in\mathbb{Z} \colon x=3k \text{ for some }k\in\mathbb{Z} \}. \end{equation*}

  2. Describe the set of all even natural numbers that are less than 100 with set-builder notation.

  3. Describe the following set in set-builder notation:

    \begin{equation*} \{ -8,-7,-6,\ldots,10\}. \end{equation*}
Solution

Worksheet 2.1.1 More Set Notation Exercises

1.

  1. Use proper notation to write an element from each of the following sets: \(\mathbb{N},\mathbb{Z},\mathbb{R},\mathbb{Q}\text{.}\)

  2. Use proper notation to write an element that is not in each of the sets above.

2.

In which of the following sets is \(-2\) an element?

  1. \(\displaystyle S_1 = \bigl\{ -1, -2, \{-1\}, \{-2\}, \{-1,-2\} \bigr\}.\)

  2. \(\displaystyle S_2 = \bigl\{ x\in\mathbb{N} \colon -x\in\mathbb{N} \bigr\}.\)

  3. \(\displaystyle S_3 = \bigl\{x\in\mathbb{Z} \colon x^2=2^x \bigr\}.\)

  4. \(\displaystyle S_4 = \bigl\{ x\in\mathbb{Z}\colon |x|=-x \bigr\}.\)

3.

Determine the cardinality of each set.

  1. \(\displaystyle A=\{0,2,4,6,\ldots,20\}.\)

  2. \(\displaystyle B=\{25,26,27,28,\ldots,75\}.\)

  3. \(\displaystyle C=\{19,23,27,\ldots,59\}.\)

  4. \(\displaystyle D=\bigl\{\, \{1,2\},\{1,2,3,4\}\, \bigr\}.\)

  5. \(\displaystyle E=\{\emptyset\}.\)

4.

For the sets below, find the designated unions and intersections. Represent the relationship in a single Venn diagram.

\begin{equation*} A_1 = \{1,3,5,7,9\} \quad A_2 = \{1,2,3,4,5\} \quad A_3 = \{2,4,6,8,10\} \end{equation*}

  1. \(\displaystyle A_1\cup A_2\)

  2. \(\displaystyle A_1\cap A_2\)

  3. \(\displaystyle A_2\cap A_3\)

  4. \(\displaystyle A_1\cap A_3\)

5.

Consider the set

\begin{equation*} S=\bigl\{ 1,2,3, \{3\}, \{1,2\},\{1,3,4\},\{1,4,5\},5,6,\{1,3,5\}\, \bigr\}. \end{equation*}

  1. Circle which of the following are elements of \(S\text{:}\)

    \begin{equation*} \emptyset \quad 1 \quad \{1\} \quad 3 \quad \{3\}\quad \{1,2,3\} \quad \{1,3,4\} \quad \{1,2,3,5,6\} \quad \{1,3,5\} \end{equation*}

  2. Circle which of the following are subsets of \(S\text{:}\)

    \begin{equation*} \emptyset \quad 1 \quad \{1\} \quad 3 \quad \{3\}\quad \{1,2,3\} \quad \{1,3,4\}\quad \bigl\{\, \{1,3,4\}\, \bigr\} \quad \bigl\{\, \{1,3,4\},5,6 \bigr\} \end{equation*}
